이스케이프

마크다운용 특수기호를 일반 문자로 표현하려면 \(역슬래시)를 사용

  • 예시1
    • & 옵시디언 플러그인
    • [[옵시디언 플러그인]]
  • 예시2
    • #Obsidian
    • #Obsidian
타이포 그래피
  • 관련 플러그인
    • [Editing Toolbar](https://github.com/PKM-er/obsidian-editing-toolbar) : 텍스트 편집 툴바 제공
    • [Style Setting](https://github.com/mgmeyers/obsidian-style-settings) : 설치된 플러그인 또는 테마 스타일 변경
  • 관련 단축키
    • ⭐ 굵게 : `Cmd + B`
    • ⭐ 기울임 : `Cmd + I`
    • ⭐ 하이라이트 : `Cmd + H`
    • ⭐ 취소선 : `Cmd + U`

커스텀 테마와 CSS로 인터페이스 조정 가능 멀티미디어 파일 처리의 한계
